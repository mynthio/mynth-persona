# Mynth Persona

## Getting Started

1. Start the database:
   ```bash
   docker-compose up -d
   ```

2. Copy environment variables:
   ```bash
   cp .env.example .env
   ```

3. Install dependencies and start the development server:
   ```bash
   pnpm install
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
